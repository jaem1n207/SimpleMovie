# Movies

> API used in project: [KOBIS OPEN API](http://www.kobis.or.kr/kobisopenapi/homepg/main/main.do "Go KOBIS")

### Introduce
* Daily/Weekly BoxOffice
* Movie Info
* Search Movie

### Project Stack
* React
* React Router
* TypeScript
* Redux
* Redux Thunk
* Container Presenter Pattern 
* Redux-logger
* Styled Components
* Immer
