# Movies

> API used in project: [KOBIS OPEN API](http://www.kobis.or.kr/kobisopenapi/homepg/main/main.do "Go KOBIS")

### Introduce
* Daily/Weekly BoxOffice
* Movie Info
* Company Info
* Cast Info (To be added)

### Project Stack
* React
* React Router
* TypeScript
* Redux
* Redux Thunk
* Redux-logger
* Styled Components
* Immer
